绑定模板数据lvs.BindTmpl
绑定页面模板数据及令牌、索引
lvs.BindTmpl( tmpl, container, token, idxid, data, okfunc() );
tmpl - 模板ID，以#开头。
container - jquery对象，绑定完成后的页面容器。
token-页面令牌
idxid-页面索引
data-需要绑定的数据
